##  Project Overview

PathWise AI is a mobile application designed to provide personalized career guidance to university students using Artificial Intelligence.

The app analyzes user data such as academic background, skills, interests, and career goals to recommend suitable career paths, identify skill gaps, and suggest learning roadmaps.

This project aims to reduce career confusion and help students make informed decisions about their future.

---

##  Key Features

- User Registration & Login (Firebase Authentication)
- Personalized Career Recommendations
- Skill Gap Analysis
- Learning Roadmap Suggestions
- AI Chatbot Career Assistant (NLP-based)
- Job Trend Insights
- Cross-platform support (Android & iOS)
